# ArbeitnehmerPowerRechenComputer

[![Live Demo](https://img.shields.io/badge/Live_Demo-Öffnen-blue)](https://deine-vercel-url.app)
[![GitHub Stars](https://img.shields.io/github/stars/juschni/arbeitnehmerpowerrechencomputer)](https://github.com/juschni/arbeitnehmerpowerrechencomputer)

**Der scharfe Rechner für echte Arbeitnehmer-Macht.**  
Strategien, Argumentationsgrundlagen und Tools für **alle** Situationen im Arbeitnehmer-Lebenszyklus – von Bewerbung bis Exit. Kein Kuschel-Coaching. Kein HR-Bullshit.

## Was er liefert
- Sofort einsetzbare, scharfe Argumentationsbausteine für jede Verhandlung
- Vollständiger Lebenszyklus-Check: Bewerbung, Gehalt, Kündigung, Mobbing, Elternzeit, Grauzonen
- Praxiserprobte Strategien + copy-paste Formulierungen (AN-seitig optimiert)
- Tabellen, Entscheidungsmatrizen, Risiko-Aufwand-Wirkung-Vergleiche

## Warum das Tool existiert
Als Arbeitnehmer verhandelst du meist allein gegen Profis.  
Dieser Rechner gleicht die Macht aus – direkt, praxisnah, ohne Floskeln.

## Quick Start
```bash
git clone https://github.com/juschni/arbeitnehmerpowerrechencomputer.git
cd arbeitnehmerpowerrechencomputer
npm install
npm run dev
```
